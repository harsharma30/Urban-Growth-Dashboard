# 🚀 UrbanPulse AI

### Predictive Real Estate Analytics Dashboard (India)

---

## 📌 Overview

UrbanPulse AI is an interactive **data-driven dashboard** designed to analyze and visualize real estate growth trends across major Indian cities.

It provides insights into:

* 📈 Growth Velocity Score (GVS)
* 🏙️ High-performing zones
* 📊 Market demand & rental absorption
* 🏗️ Government infrastructure impact
* 🔮 Future price forecasts

---

## ✨ Features

* 🗺️ **Interactive Heatmap**
  Visual representation of growth zones across India

* 📊 **Zone Analytics Panel**
  Detailed breakdown of:

  * Municipal score
  * Market demand
  * Price trends
  * Rental absorption

* 📈 **Growth Velocity Score (GVS)**
  A custom metric to evaluate real estate potential

* 🧠 **Market Insights Dashboard**

  * Top-performing zones
  * City-wise analysis
  * Trend indicators

* 📥 **Data Ingestion Simulation**
  Simulates real-time data sources like:

  * Market platforms
  * Government portals

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Custom CSS (inline + design tokens)
* **Visualization:** SVG-based custom heatmap
* **Deployment:** Vercel

## ⚙️ How It Works

1. Data for multiple zones is defined (mock dataset)
2. Each zone is assigned a **Growth Velocity Score (GVS)**
3. Heatmap dynamically visualizes zones based on performance
4. Sidebar provides detailed analytics and forecasts
